# タスク完了時のチェックリスト

## 必須チェック項目

### 1. 型チェック
```bash
cd src
yarn tsc --noEmit
```
- 型エラーがないことを確認

### 2. Lintチェック
```bash
cd src
yarn lint
```
- ESLintエラーがないことを確認
- 自動修正は使用せず、手動で修正すること

### 3. テスト実行
```bash
cd src
yarn vitest run
```
- すべてのテストがパスすることを確認

### 4. ビルド確認
```bash
cd src
yarn build
```
- プロダクションビルドが成功することを確認

## 推奨チェック項目

### 開発サーバーでの動作確認
```bash
cd src
yarn dev
```
- ブラウザで動作確認
- コンソールエラーがないことを確認

### コードレビュー
- 既存パターンに準拠しているか
- クリーンアーキテクチャの原則に従っているか
- 型安全性が保たれているか

- [ ] ビルドが成功
- [ ] 開発サーバーで動作確認済み
